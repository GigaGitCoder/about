Обо мне

Привет, я Igor Xmel — backend-разработчик из Ростова-на-Дону. Пишу на Python и Java, строю серверные системы, REST API, микросервисы и телеграм-ботов под самые разные задачи.

Особое место занимает разработка ботов — aiogram, telebot, FSM-сценарии, интеграции с внешними сервисами. Реализовал ботов разной сложности: от простых уведомлений до полноценных CRM-инструментов.

Активно участвую в хакатонах и соревнованиях — люблю работать под дедлайном и решать нетривиальные задачи в команде. Большинство рабочих проектов — в закрытых репозиториях.

02

Специализация

Разработка ботов

Телеграм-боты на aiogram и telebot — FSM, inline-меню, платёжные интеграции, вебхуки. Боты для бизнеса и личных проектов.

Backend & API

REST API на FastAPI и Flask, микросервисы на Spring Boot. JWT-авторизация, Pydantic-валидация, Swagger UI, чистая архитектура.

Базы данных

PostgreSQL, MySQL, SQLite — проектирование схем, миграции (Alembic, Liquibase). ORM: SQLAlchemy, Hibernate. Кэширование: Redis.

DevOps & Deploy

Docker и docker-compose для любого стека. CI/CD через GitHub Actions. Развёртывание на Linux-серверах под Windows / Mac / Linux.
